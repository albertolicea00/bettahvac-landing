# BettaHVAC Landing Page

This is the repository for the **BettaHVAC** landing page, an HVAC (Heating, Ventilation, and Air Conditioning) services company.

The project is built using [Svelte](https://svelte.dev/) and [Vite](https://vitejs.dev/), offering a fast development experience and a highly optimized website. All dynamic content on the site is managed through a JSON configuration file.

## Features

- 🚀 **Modern Tech Stack**: Built with Svelte 5 and Vite.
- 🎨 **Custom Design**: Uses the company's brand colors (blue, red, and gold).
- 📝 **Simple Content Management**: All content (text, contact information, services) is modular and driven by `src/lib/content.json`.
- ⭐ **Review System**: Modular integration to display both internal site reviews and external reviews (e.g., Google Reviews).

## Prerequisites

Before you begin, ensure you have the following installed on your system:

- [Node.js](https://nodejs.org/) (version 18 or higher recommended)
- `npm` (included with Node.js), `pnpm`, or `yarn`

## Installation and Setup

Follow these steps to get the project running on your local machine:

1. **Clone or download this repository** (or navigate to the project directory if you already have it locally):

   ```bash
   cd "bettahvac landing"
   ```

2. **Install dependencies**:

   ```bash
   npm install
   ```

3. **Start the development server**:

   ```bash
   npm run dev
   ```

4. **Open in your browser**:
   By default, Vite will start the server at `http://localhost:5173`. Open that link in your browser to view the page and see changes in real-time (Hot Module Replacement enabled).

## Available Commands

Within the project directory, you can run the following commands:

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the optimized application for production. The generated files will be saved in the `dist/` folder.
- `npm run preview`: Starts a local server to preview the production build generated by `npm run build`.

## Main Project Structure

- `src/App.svelte`: The main Svelte application component.
- `src/lib/content.json`: File containing all the text and content of the page for easy editing without touching the source code.
- `src/lib/components/`: Folder containing the various modules and visual sections of the page (Nav, Contact, Reviews, etc.).
- `src/lib/ReviewService.js`: Service responsible for handling and synchronizing the review logic.
